# 🎮 Tic Tac Toe – チーム開発課題

これは初心者向けチーム開発の一環として作成した **Tic Tac Toe（〇✕ゲーム）** です。  
HTML / CSS / JavaScript を使って、ブラウザ上で遊べる三目並べゲームを実装しています。

---

## 🔧 機能一覧

- 2人対戦モード（プレイヤーXとプレイヤーO）
- 勝利判定と引き分け判定
- 勝者が表示されるステータスメッセージ
- リセットボタンでゲームの再スタートが可能
